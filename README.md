# Историческая карта Таганрога

Интерактивная карта города: 33 достопримечательности, сгруппированные по пяти историческим эпохам. По клику на точку открывается карточка с фото, адресом и историческим описанием. Эпохи можно включать и выключать галочками слева.

Технологии: HTML + Leaflet + OpenStreetMap. Ничего устанавливать не нужно — открывается в любом браузере.

---

## Как добавить фотографии

1. Положите фото каждого места в папку `images/`.
2. Назовите файлы **ровно** так, как указано в `images/СПИСОК_ФОТО.txt` (например, `04-pamyatnik-petru.jpg`).
3. Формат — `.jpg`. Если файла нет, карточка просто покажется без фотографии — сайт не сломается.

Совет: бесплатные фото без проблем с авторскими правами удобно брать на Викискладе (commons.wikimedia.org). Не забудьте сослаться на источник в самом проекте, если этого требует учитель.

## Как выложить на GitHub и открыть по ссылке

1. Создайте на GitHub новый репозиторий (кнопка **New**), название например `taganrog-map`, поставьте галочку **Public**.
2. Нажмите **Add file → Upload files** и перетащите туда `index.html`, папку `images/` и этот `README.md`. Нажмите **Commit changes**.
3. Откройте вкладку **Settings → Pages**.
4. В разделе **Build and deployment** в поле **Source** выберите **Deploy from a branch**, ветку **main**, папку **/ (root)**, нажмите **Save**.
5. Подождите 1–2 минуты и обновите страницу — вверху появится ссылка вида
   `https://ВАШ-ЛОГИН.github.io/taganrog-map/`
6. Эта ссылка и есть готовый сайт — её можно отправить кому угодно.

## Если захочется доработать

- **Тексты и точки** редактируются в `index.html` — найдите строку `const DATA =` в конце файла: там список всех объектов (название, координаты, адрес, описание, путь к фото).
- **Цвета эпох** — там же, в начале списка `epochs`.
- Координаты указаны в десятичных градусах и взяты из исходного файла с описаниями. У нескольких объектов в источнике они были приблизительными — при желании их можно уточнить через Яндекс- или Google-карты.

Два объекта из отчёта («Обелиск металлургам» и «индустриально-технологические памятники») не добавлены, потому что в источнике у них не было конкретных координат. Если найдёте точки — их легко дописать в список `DATA` по образцу остальных.
